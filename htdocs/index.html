HLmaps is a Perl/CGI script for Half-Life game servers running on Linux, FreeBSD, and WinNT. It allows web site visitors to view all your loaded maps, screen shots, download links and some file statistics in a nice, sortable table.

HLmaps_php is a new rewrite by Brian Porter - check it out below!

The latest version of HLmaps_php is 0.9 (03/18/2002). (New!!)
The latest version of HLmaps is 1.1 (03/27/2001). (The "original" flavor)
The latest version of HLmaps_images is 3.0 (01/31/2002).

(Users of csserverstats, halfStats, hlds_ld and hlmon will want at least HLmaps_images)

All can be found on the downloads page. Before grabbing them, it's a good idea to examine the README, CHANGELOG and INSTALL files. Developers will want to read the TODO and CONTRIBUTING files as well.

Features:

  • Extremely customizable output, including support for almost any spoken language and color scheme, all done using a straightforward configuration file.
  • Very low run-time CPU utilization. Data collection is done via a script which runs during off-hours.
  • Cross platform compatability with Linux, FreeBSD and NT.
  • Perl's "strict", "warn" and "taint" modes are fully employed for safety and Apache's mod_perl/PerlRun compatability mode works great with HLmaps.
  • Each table column is sortable in ascending and descending order by simply clicking on the header.
  • Optional MySQL database integration (see INSTALL.MySQL for details) is included.
  • Great for clans to show maps loaded on a server, file details, download links, txt files and more.
Requirements: Linux, FreeBSD or WinNT, Perl 5.x, a web server supporting CGI, Valve's Linux Half-Life server program and optionally the MySQL database engine.
HLmaps was written by Scott McCrory (HLmaps_php by Brian Porter) and is released as open-source software under the GNU Public License. If you find it useful then I ask that you contribute back to this project in some way. In particular, I'd really appreciate help with collecting good quality map images for all of the major Half-Life mods (like Counterstrike, TFC, Firearms, Action, Phineas, Valve, etc.). Please read the CONTRIBUTING file to find out how to help and while you're at it take one of the surveys to let me know what you'd like to see in the next release.
